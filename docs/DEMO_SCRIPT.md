# LogSage AI Demo Script

## Preparation
1. Ensure Ollama is running in the background (`ollama serve`).
2. Run the Streamlit application: `streamlit run app.py`.

## Step 1: Introduction
- "Welcome to LogSage AI. This tool is designed to save on-call engineers hours of manual log parsing."
- Point out the sidebar showing **"🟢 Ollama Online"**. Explain that all AI processing happens locally and securely without sending sensitive logs to the cloud.

## Step 2: Running an Analysis
- Navigate to **"🏠 Analyze Logs"**.
- Click "Browse files" and select `sample_logs/database_error.log`.
- Click the **"🚀 Run Analysis"** button.
- *Watch the progress bar update through the 7 agent steps.*

## Step 3: Reviewing the Dashboard
- Once complete, show the **📊 Dashboard** tab.
- Highlight the 4 metric cards (Total Lines, Errors Found, Max Severity, Avg Severity).
- Read the **Executive Summary** generated by the AI.
- Show the interactive Plotly charts.

## Step 4: Inspecting Anomalies
- Switch to the **🔍 Anomalies** tab.
- Expand one of the anomalies.
- Highlight how the AI identified the **Root Cause** and provided actionable **Remediation Steps**.
- Show the raw **Context** block that the AI used to make its decision.

## Step 5: Exporting the Report
- Switch to the **📄 Report** tab.
- Click **"⬇️ Download PDF Report"**.
- Open the downloaded PDF to show the clean, executive-ready formatting.

## Step 6: Incident History
- Navigate to **"📜 Incident History"** in the sidebar.
- Show that previous incidents are saved securely in SQLite.
- Search for the recent run to demonstrate search capability.
